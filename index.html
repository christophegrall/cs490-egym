Synopsis

This project is for CSCI 490 - Software Engineering II at Coastal Carolina University. The intent of this project is to be a health tracking app for individuals, trainers, and other health improvers. The core technology used in this project is the Java Spring framework, AngularJS 2, Bootstrap 3 for design, and more.
